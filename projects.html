Business Feasibility Prediction System (BFPS)

A comprehensive ML system for predicting small-business success

The Business Feasibility Prediction System is a large-scale machine learning platform that predicts the success of small businesses using multi-source data integration and advanced predictive modeling. The system achieves 89% accuracy in success classification and provides actionable insights for business planning.

Data Sources

  • U.S. Census ACS - Demographic and economic data
  • Bureau of Labor Statistics (BLS) - Employment and wage statistics
  • NOAA Climate Data - Weather patterns and seasonal trends
  • Yelp Business + Review Data - Business performance and customer sentiment

System Components

Data Pipeline (ETL)

Automated extraction, transformation, and loading of data from multiple sources. Handles data cleaning, normalization, and integration of heterogeneous datasets.

Python Pandas API Integration Data Cleaning

Feature Store

Centralized repository for engineered features including demographic indicators, economic metrics, climate patterns, and business performance indicators.

Feature Engineering SQLite Data Versioning

ML Models

Multi-model ensemble approach combining classification, regression, and time-series forecasting:

  • XGBoost Classifier - Success prediction (89% accuracy)
  • Regression Models - ROI and payback period estimation
  • Time-Series Analysis - Seasonality detection and forecasting
XGBoost Scikit-learn Hyperparameter Tuning Cross-Validation

Dashboard

Interactive visualization platform for exploring predictions and insights. Provides business owners with actionable recommendations.

Tableau React D3.js Interactive Viz
